# ⚡ EV Population Data Dashboard

## 📊 Project Overview
This project focuses on analyzing the **Electric Vehicle (EV) population data** to explore adoption trends, manufacturer dominance, and regional distribution of EVs across the United States.  
A **Tableau Dashboard** was created to visualize and interpret the insights effectively.

---

## 🎯 Objectives
- Analyze the **growth and adoption** of electric vehicles.  
- Identify **top-performing manufacturers and models**.  
- Examine **regional EV distribution** and **vehicle types**.  
- Support **data-driven decision-making** in EV policies and infrastructure planning.

---

## 🧰 Tools & Technologies
- **Tableau** – for data visualization and dashboard creation  
- **Microsoft Excel** – for data preparation and cleaning  
- **GitHub** – for version control and project sharing  

---

## 📂 Repository Contents

---

## 📈 Key Insights
- Significant **growth trend** in EV registrations in recent years  
- **Top EV manufacturers and models** identified  
- Regional patterns highlight **high adoption in urban and coastal areas**  
- Comparison between **Battery Electric Vehicles (BEVs)** and **Plug-in Hybrid Electric Vehicles (PHEVs)**  

---

## 🚀 How to Use
1. Download the `EVP Data Tableau.twb` file.  
2. Open it using **Tableau Desktop** or **Tableau Public**.  
3. Interact with the dashboard filters and charts to explore the data.

---

## 📚 Dataset Information
The dataset used in this project represents **U.S. Electric Vehicle Population Data** and includes:  
- Vehicle make, model, and model year  
- Electric range and vehicle type (BEV or PHEV)  
- Base MSRP  
- Clean alternative fuel vehicle eligibility  
- Regional and city-level information
